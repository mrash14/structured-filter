Structured-Filter

Structured-Filter (currently v3.0.0) is a Web UI for building structured search queries. It implemented on jQuery and Materializecss.

With it you can build structured search conditions like Firstname starts with 'A' and Birthday after 1/1/1990 and State in (CA, NY, FL)...

API Tests: Add Condition - Remove First Condition - Number of Conditions - Clear

Value Formats: JSON - Text - URL


Download and documentation are available at GitHub.
